### v1.2.0: 다중 페이지 전술판 시스템 🎯 NEW (업데이트됨)

**핵심 신규 기능: 다중 페이지 기반 패턴 연습**
- [ ] **다중 페이지 전술판**: 기존 전술판을 여러 페이지로 확장
- [ ] **페이지 네비게이션**: 탭 기반 1단계→2단계→3단계 전환
- [ ] **페이지별 보드 상태**: 각 페이지마다 독립적인 선수/드로잉 배치
- [ ] **패턴 라이브러리**: 다중 페이지 패턴 저장/불러오기
- [ ] **기본 템플릿**: 5가지 일반적인 연습 패턴 프리셋

**UI 구성:**
```
[1단계] [2단계] [3단계] [+새페이지]
─────────────────────────────────
    기존 전술판 인터페이스
─────────────────────────────────
[← 이전] [다음 →] [패턴저장]
```

**핵심 장점:**
- **기존 UX 유지**: 사용자가 익숙한 전술판 인터페이스 그대로
- **복잡성 해결**: 별도 편집기 없이 기존 시스템 확장
- **직관적 플로우**: 페이지 넘기기로 패턴 흐름 이해 쉬움
- **점진적 구현**: 기존 코드 최대 활용으로 빠른 개발

**사용 시나리오 예시:**
- "2:1 패스로 수비 따돌리기" (3페이지)
  - 1페이지: 초기 배치
  - 2페이지: 첫 패스 후 위치
  - 3페이지: 돌파 완료 상황
- "리턴패스 후 침투" (2페이지)
- "오버랩 연계" (4페이지)

**예상 개발 기간:** 1-2주 (기존 시스템 활용으로 단축)
